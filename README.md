# Yamibo Scraper（百合会小说抓取工具）

一个用于抓取百合会论坛连载帖内容的小工具，可导出为 **TXT** 或 **EPUB**，便于离线阅读。

## 环境准备

项目使用 [uv](https://github.com/astral-sh/uv) 管理依赖。

### 1) 安装 uv

```bash
pip install uv
```

### 2) 安装依赖

```bash
uv sync
```

## 配置方式（统一文件管理）

程序会把配置存到项目根目录的 `yamibo_config.json`，包括：

- `user_agent`
- `cookie`
- `username`
- `password`
- `book_title` / `book_author`
- `raw_html_catalog`

首次运行可直接通过 CLI 输入，输入后会自动保存。后续需要修改时，在菜单里选择 **“修改配置”** 即可。

### raw_html_catalog 推荐录入方式

在“修改配置 -> 7. raw_html_catalog”里支持三种方式：

1. 单行粘贴
2. 多行粘贴（`END` 单独一行结束）
3. 从本地文件读取

程序会提示“字符长度 + 检测到的链接数”，方便你判断是否读入成功。

## 运行

- `auth.py`：登录与会话初始化
- `search.py`：关键词搜索帖子
- `cli.py`：终端交互
- `config_store.py`：本地配置加载与保存

说明：搜索后“自动解析目标帖子目录”还在开发中，当前会回退到 `raw_html_catalog`。

## 常见问题

### 1) 为什么还有 Cookie 登录？

账号密码登录是默认推荐方式；保留 Cookie 模式是为了兼容验证码、风控或页面结构变化等情况。

运行后流程：

1. 显示终端编码提示（中文乱码排查）
2. 主菜单：
   - 开始抓取
   - 修改配置
   - 退出
3. 开始抓取时再选择：
   - 输出格式（TXT / EPUB / 同时）
   - 登录方式（账号密码 / Cookie）
   - 抓取来源（`raw_html_catalog` / 关键词搜索）

输出文件默认保存在 `output/` 目录。
