# 📚 Yamibo Scraper (百合会小说抓取工具)

这是一个轻量级的 Python 爬虫项目，用于将百合会（Yamibo）论坛上的连载小说抓取到本地，并支持一键生成 **TXT** 或排版精美的 **EPUB** 电子书格式，方便使用阅读器进行离线阅读。

## 🛠️ 快速开始

本项目使用现代化的 Python 包管理器 [uv](https://github.com/astral-sh/uv) 进行极速环境构建。

### 第一步：安装 uv

如果你还没有安装 `uv`，但是你已经有了 Python(pip) 环境，最简单、最友好的安装方式就是直接使用 pip：

```bash
pip install uv
```

下面是独立安装的方法（也就是不需要通过PyPI），参考[uv-zh-cn](https://hellowac.github.io/uv-zh-cn/getting-started/installation/#uv)

- **Windows:**
  ```powershell
  powershell -ExecutionPolicy ByPass -c "irm [https://astral.sh/uv/install.ps1](https://astral.sh/uv/install.ps1) | iex"
  ```
- **macOS / Linux:**
  ```bash
  curl -LsSf [https://astral.sh/uv/install.sh](https://astral.sh/uv/install.sh) | sh
  ```

### 第二步：获取项目并准备环境
克隆或下载本项目到本地，进入项目文件夹后，运行以下命令，`uv` 会自动为你创建虚拟环境并瞬间安装好所有依赖：

```bash
uv sync
```

## 🔑 核心配置教程 (小白必看)

在使用之前，你必须在 `yamibo_scraper.py` 文件中配置三个东西：**Cookie**、**User-Agent** 和 **目录 HTML**。

### 1. 如何获取 Cookie 和 User-Agent？

1. 使用电脑浏览器（推荐 Chrome/Edge）登录 [百合会论坛](https://bbs.yamibo.com/)。
2. 登录成功后，按下键盘上的 **`F12`** 打开“开发者工具”。
3. 点击开发者工具顶部的 **`网络 (Network)`** 选项卡。
4. 按 **`F5`** 刷新当前网页。
5. 在左侧的网络请求列表中，找到并点击第一个文件（通常叫 `forum.php` 或当前网址的名称）。
6. 在右侧面板往下划，找到 **`请求标头 (Request Headers)`** 区域：
   - 找到 `Cookie:`，将后面那一长串乱码（直到换行前）全部复制。粘贴到代码中的 `YOUR_COOKIE_STRING` 处。
   - 找到 `User-Agent:`，将后面的内容复制。粘贴到代码中的 `YOUR_USER_AGENT` 处。

### 2. 如何获取目录 HTML (RAW_HTML_CATALOG)？

为了精准抓取而不漏章，我们需要提取楼主发布的目录大纲（没有目录的情况比较少见，可以尝试修改抓取条件为只抓取作者）。

1. 在浏览器中打开你想爬取的小说目录页（通常是连载帖的第一页，楼主的首层楼）。
2. 在网页的空白处**点击鼠标右键**，选择 **【查看网页源代码】**（或使用快捷键 `Ctrl+U`）。
3. 这时会打开一个写满密密麻麻代码的新页面。按下 **`Ctrl+F`** 打开搜索框。
4. 在搜索框输入 `table` 可以查找到目录类 ，迅速定位到目录代码开始的地方。
5. 从第一章所在的 `<a href="...">` 标签开始往下拖动鼠标，一直高亮选中到最后一章结束。
   *(💡 提示：连着外层的 `<br>` 或 `<div>` 一起复制也没关系，程序的正则引擎会自动清洗。)*
6. **复制**。
7. 打开 `yamibo_scraper.py`，找到 `RAW_HTML_CATALOG = """ ... """` 区域，将你复制的代码**原封不动地粘贴到三个引号之间**，完全覆盖掉里面的示例代码。

## 🚀 运行爬虫

完成上述配置后，在项目目录下运行：

```bash
uv run yamibo_scraper.py
```

终端会弹出交互提示，询问你是想仅生成 TXT 还是同时生成 EPUB。输入对应的数字回车，然后去喝杯茶，等待进度条跑完即可！

生成的文件会自动保存在项目目录下的 `output/` 文件夹中。
